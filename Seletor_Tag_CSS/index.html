Agrupamento de seletores

Podemos aplicar as regras CSS para vários seletores diferentes.Para isso, podemos separar os seletores utilizando uma vírgula (,)

Assim, não precimos ficar copiando e colando os mesmo códigos para todos, nós podemos fazer um só bloco e reutilizar as mesmas regras!
